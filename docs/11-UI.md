# 第一章 设计理念

## 1.1 产品定位

SmartGrade 是面向学校教师和管理人员的数字化工作平台。

核心用户：

- 班主任
- 年级主任
- 政教
- 宿舍管理员
- 系统管理员

UI设计目标：

让教师能够：

快速查看

快速处理

快速反馈

减少操作成本。

---

# 第二章 设计原则

## 2.1 简洁优先

避免复杂视觉。

减少：

- 装饰
- 动画
- 无意义信息

突出：

- 状态
- 任务
- 操作入口

---

## 2.2 信息优先

学校管理场景：

信息密度高。

设计优先级：

信息展示

>

操作效率

>

视觉效果

---

## 2.3 状态明确

所有业务状态必须视觉化。

例如：

请假状态：

🟢 在校

🟡 已批准待离校

🟠 已离校
