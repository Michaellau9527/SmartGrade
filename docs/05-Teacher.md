# SmartGrade 教师端设计规范（Teacher Design Specification）

Version：1.0

---

# 第一章 教师端设计原则

教师端是 SmartGrade 的核心使用端。

设计目标：

- 简单
- 高效
- 少输入
- 少跳转
- 一致性

设计原则：

① 三次点击完成高频操作

② 页面层级不超过三级

③ 一个页面只完成一个核心任务

④ 所有按钮位置保持一致

⑤ 所有列表支持搜索

⑥ 所有状态使用统一颜色

---

# 第二章 导航结构

采用：

底部 Tab + 页面跳转。

```
┌──────────────────────────┐
│        工作台            │
├──────────────────────────┤
│ 通知 │ 待办 │ 我的 │
└──────────────────────────┘
```

工作台负责：

所有业务入口。
